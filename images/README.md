## Image Requirements

### Project Images
- **Format**: JPG or PNG
- **Size**: 800x500px (recommended)
- **Quality**: High resolution for crisp display
- **Naming**: Use descriptive names (homepage.jpg, services.jpg, etc.)

### Profile Image
- **Format**: JPG or PNG
- **Size**: 300x300px (square)
- **Quality**: Professional headshot

## How to Add Your Images

1. **Take Screenshots** of your actual projects
2. **Resize** them to 800x500px using any image editor
3. **Save** them in the appropriate project folders
4. **Replace** the placeholder URLs in `script.js`

## Quick Setup

1. Add your profile photo as `images/profile/joylene.jpg`
2. Add 3 screenshots for each project in their respective folders
3. The website will automatically load your images!

## Image Optimization Tips

- Use **WebP format** for better compression (optional)
- Keep file sizes under **500KB** for faster loading
- Use **descriptive alt text** for accessibility
- Consider using **lazy loading** for better performance
